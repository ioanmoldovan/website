Privacy Policy

- Apps

No personal data is collected by any Android app developed by "Ioan Moldovan".

Also, No analytics are used, No trackers, No background services. Nothing that could affect user experience in a negative fashion.

Data is stored only on device or at Google (if the user opted to sync application data to Google in Android's Backup settings).

- Website

In order to prevent abuse of ioanm.ro, the following anonymous data is collected:

  • IP Address
  • User Agent
  • Requested URL, HTTP Response code (Errors, if any)

This data can not be used to identify a person and collecting it is standard practice.
